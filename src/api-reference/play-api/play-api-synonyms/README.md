# Synonyms

## About this endpoint
With this endpoint, you can manage all your synonyms, creating new ones or modifying existing ones.

**Base URL**

`https://api.empathybroker.com/play/v1/INSTANCE_ID/synonyms`

## Synonyms Endpoints
* Get Synonym
* Create Synonym
* Update Synonym
* Delete Synonym
* Enable Synonym
* Disable Synonym
