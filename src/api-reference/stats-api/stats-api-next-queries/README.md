# Next Queries Reports

## About these reports
This set of reports return all relevant pairs of queries (tuples q1 - q2) in a given date range. These pairs represent the most repeated behaviour in the site and are useful to understand user's journey.

## Business value
Which insights can be gathered from these reports?

* **Understanding user journey:** Which are the most pair of queries repeated on your site?
  ![next-queries-report](/docs-empathy/assets/media/docs-next-queries-report.png)
* **Understand search refinement:** What do users search after a search without results? Which search terms follow others?
