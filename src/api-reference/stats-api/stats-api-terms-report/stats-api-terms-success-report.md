# Term Success

## About this report
This report returns a list of top search terms and its main KPIs. 
      
If terms are specified in the corresponding parameter, only the queries that include those terms are returned

## Request
This reports is requested for a date range and optionally narrowed down for a set of filter values. 
  
The _terms_ parameter can be used to restrict the outpput to queries coanting such a term.

**GET** `https://api.empathybroker.com/stats/v2/analysis/{Instance_ID}/TermSuccess`

**Query Parameters**


  | PARAMETER             |   TYPE  |
  |-----------------------|:-------:|
  | [start_date](/api-reference/stats-api/stats-api-query-parameters-glossary/#tagging-input-parameters-glossary-common)            |Int (ISO 8601 format)|
  | [end_date](/api-reference/stats-api/stats-api-query-parameters-glossary/#tagging-input-parameters-glossary-common)              |Int (ISO 8601 format)|
  | [filters](/api-reference/stats-api/stats-api-query-parameters-glossary/#tagging-input-parameters-glossary-filters)               |list[str] |
  | [type](/api-reference/stats-api/stats-api-query-parameters-glossary/#stats-report-specific-params)               |int |
  | [terms](/api-reference/stats-api/stats-api-query-parameters-glossary/#stats-report-specific-params)               |str |

**Bearer Token** [Get your token](/api-reference/stats-api/#stats-api-stepbystepguide)

**Example request:** 
`https://api.empathybroker.com/stats/v2/analysis/ebdemo/TermSuccess?start_date=2019-07-10&end_date=2019-07-17&filters=lang,scope&lang=es_ES&scope=desktop`
