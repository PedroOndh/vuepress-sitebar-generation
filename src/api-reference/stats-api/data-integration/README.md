# Data Integration

## Overview
Data integration is the process of combining data from multiple source systems to create unified sets of information for both operational and analytical uses. 

Powerful tools like **Google Data Studio Connector** provided to retrieve the information that is indexed in the Empathy Servers to access data from different sources to complete transactions, make easy and visual report and other tasks.
